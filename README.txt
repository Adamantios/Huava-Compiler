------------------------------------------------------- INFO --------------------------------------------------------

- Έχουμε κάνει μία παραλλαγή,
  ώστε ο χρήστης να μπορεί να δώσει ως είσοδο πολλά αρχεία τα οποία θα αποτελούν ένα πρόγραμμα.

- Τα αρχεία εισόδου πρέπει να έχουν κατάληξη .huava και τα αρχεία εξόδου είναι τύπου .huaclass.

- Υπάρχει δυνατότητα παραγωγής των κλάσεων, εισάγοντας μετά την προεραιτική επιλογή --encoding 
  και πριν τα αρχεία εισόδου, την προεραιτική επιλογή --generate.
  Μετά την εκτέλεση του προγράμματος, οι κλάσεις βρίσκονται στο φάκελο huaclasses.

- Έχουμε δημιουργήσει κάποια αρχεία για τον έλεγχο του Compiler στο φάκελο FinalTestFiles.


------------------------------------------------------- USAGE -------------------------------------------------------




1. Install maven


2. Compile using maven

mvn package


3. Execute

 java -jar target/compiler-0.0.2.jar input-filename

        
or 

java -cp target/compiler-0.0.2.jar Compiler input-filename
